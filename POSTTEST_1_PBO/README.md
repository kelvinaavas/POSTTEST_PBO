
OUTPUT PROGRAM

Program dijalankan melalui terminal atau console Java. Saat program berjalan, sistem menampilkan menu antrian pelanggan barbershop. Pengguna dapat menambah pelanggan, memanggil pelanggan berikutnya, melihat daftar antrian, dan keluar dari program.

Contoh tampilan saat program dijalankan:

=== SISTEM ANTRIAN BARBERSHOP ===
1. Tambah pelanggan ke antrian
2. Panggil pelanggan berikutnya
3. Lihat daftar antrian
4. Keluar
Pilih menu: 1
Masukkan nama pelanggan: Andi
Pelanggan Andi berhasil masuk ke antrian

PENJELASAN HASIL PROGRAM

Program menjalankan sistem antrian menggunakan konsep queue.
Pelanggan masuk ke antrian sesuai urutan kedatangan.
Pelanggan pertama yang masuk dipanggil lebih dahulu.
Program menampilkan daftar pelanggan yang masih menunggu.
